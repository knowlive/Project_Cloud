*ELB, EIP 할당

1. 인스턴스 구성 -> elb 넣기
2. 보안그룹구성 ->  ssh, https, mysql(aurora), http, icmp ipv4
(경고뜰 시에 소스 위치무관 바꾸기)
3. 탄력적 ip 할당(네트워트 인터페이스로도 할당)
4. 각 tomcat 탄력적 ip를 route53 내용에 넣어주기
5. 접속되는지 탄력적 ip로 웹에 접속. 접속되면 성공

*로드밸런싱

1. 로드 밸런싱의 대상그룹 넣기.
2. 대상 그룹 생성 (이름 넣고 생성)
3. 아래 대상탭으로 들어가서 편집 클릭
4. 편집해서 원하는 tomcat 위로 올리고 저장.
5. 로드 밸런서 생성, http, https 생성
6. 리스너 https도 추가.
